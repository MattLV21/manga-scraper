MANGA SELECTORS FOR ASURA SCANS
==================================

Based on analysis of the HTML structure from Asura Scans.

NOTE: The HTML appears to be minified with escaped quotes (&quot;).
Using text content selectors or class-based selectors that remain
consistent across minified pages.

=== MANGA TITLE ===
Selector: h1:contains("The Extras Academy Survival Guide")
Alternative: h1[class*="font-semibold"]

=== MANGA COVER IMAGE ===
Selector: img[alt="Asura Scans"][class*="rounded-full"]
Alternative: a img[class*="w-12 h-12"]

=== MANGA DESCRIPTION ===
The description text appears within meta tags, but the content itself
might be rendered dynamically. Try:
Selector: meta[property="og:description"]
For rendered content, it may be in a div with summary class

=== CHAPTER LINKS ===
From the data found, chapters appear in a JSON-like structure.
Looking for links with "chapter" in URL.

Selector: a[href*="/chapter/"]

From the network data, the structure appears to be:
- Chapter links in a list/grid container
- Each link has href="/chapter/{number}"
- Chapter numbers range from 1 to 99 (based on sample data)

The selector for chapter links would be:
Selector: ul.chapter-list a[href*="/chapter/"]
Or: div.chapter-info a[href*="/chapter/"]
Or: .chapters li a[href*="/chapter/"]

If no specific container class exists, try:
Selector: a[href*="/chapter/"]:not([href*="/chapter/0"])

=== PUBLISHED STATUS INDICATOR ===
Based on the "Latest Chapter" span found, there may be a published status
selector. Look for:
Selector: span:contains("Latest Chapter")
Or: .published:contains("Published")
Or: .latest-chapter a[href="/"] (home page link when not published)

=== RECOMMENDED SELECTORS ===
{
  "mangaTitle": "h1:contains(\"The Extras Academy Survival Guide\")",
  "coverImage": "a img[class*=\"w-12 h-12 flex-shrink-0\"]",
  "mangaDescription": "meta[property=\"og:description\"]",
  "chapterLinks": "a[href*=\"/chapter/\"]:not([href*=\"/\"])>",
  "isPublished": "span:contains(\"Latest Chapter\")",
  "publishedLabel": "span:contains(\"Published\")"
}
