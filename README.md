# Manga Scraper Project

A manga reading application with web scraping capabilities.

## Features

- Scrape manga from various scanlation sites
- FastAPI backend with SQLite database
- Web frontend with React + TypeScript + Vite
- Image proxy for privacy and performance

## Tech Stack

### Backend
- **Python** with **FastAPI**
- **SQLite** database
- **Playwright** for scraping

### Frontend
- **React** with **TypeScript**
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **React Router** for navigation

## Getting Started

### Backend
```bash
pip install -r requirements.txt
uvicorn api:app --reload
```

### Frontend
```bash
cd web
npm install
npm run dev
```
